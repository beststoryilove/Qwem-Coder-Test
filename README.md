# Harmonia - 音乐播放器

一个美观易用的在线音乐播放器，支持歌词显示、播放列表管理和多主题切换。

## ✨ 功能特性

### 🎵 核心功能
- **音乐搜索与播放** - 支持搜索和播放网易云音乐曲库
- **智能歌词显示** - 支持普通歌词和逐字歌词（YRC）
- **歌词翻译** - 支持中日/中韩/中英等跨语言歌词翻译
- **播放列表管理** - 侧边栏播放列表，支持搜索、排序（A-Z / Z-A）
- **MV 播放** - 支持解析并播放歌曲的官方 MV

### 🎨 界面与体验
- **多主题切换** - 支持深色/浅色主题
- **动态岛（Dynamic Island）** - 灵动岛式交互体验
- **3D 专辑封面效果** - 专辑封面随鼠标移动产生 3D 视差效果
- **桌面歌词** - 支持桌面悬浮歌词显示
- **响应式设计** - 适配桌面端和移动端

### ⚙️ 设置选项
- **歌曲翻译开关** - 开启/关闭歌词自动翻译
- **逐字歌词开关** - 开启/关闭逐字歌词模式
- **网易云代理配置** - 自定义代理地址解决 CORS 问题
- **专辑特效开关** - 开启/关闭 3D 专辑封面动画

## 🚀 使用方式

1. 直接在浏览器中打开 `index.html` 文件
2. 在搜索框中输入歌曲名称或歌手进行搜索
3. 点击搜索结果即可开始播放
4. 使用顶部菜单栏切换主题、查看设置或管理播放列表

## 📋 技术栈

- **原生 HTML5/CSS3/JavaScript** - 无需构建工具，开箱即用
- **Font Awesome** - 图标库
- **Apple Music 风格设计** - PingFang SC / SF Pro 字体

## 🎯 快捷操作

| 功能 | 说明 |
|------|------|
| 🔍 搜索 | 顶部搜索框输入关键词 |
| 🎨 主题切换 | 点击主题按钮切换深浅色模式 |
| 📝 歌词切换 | 点击歌词按钮显示/隐藏歌词 |
| 📋 播放列表 | 点击菜单按钮打开侧边栏 |
| ⚙️ 设置 | 点击设置按钮配置播放选项 |
| 🎬 MV 播放 | 点击 MV 按钮尝试播放歌曲 MV |

## 💡 提示

- 部分功能（如歌词翻译、逐字歌词）依赖网易云 API，可能需要配置代理
- 3D 专辑封面效果在频繁触发时可能影响性能，可根据设备性能选择开启/关闭
- MV 播放功能取决于网易云是否提供该歌曲的 MV 资源
